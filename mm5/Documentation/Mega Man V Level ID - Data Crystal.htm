Level ID is used to define the numeric representation for a level in Mega Man 5.

List of Level ID's

  • 0: Gravityman's Stage
  • 1: Waveman's Stage
  • 2: Stoneman's Stage
  • 3: Gyroman's Stage
  • 4: Starman's Stage
  • 5: Chargeman's Stage
  • 6: Napalmman's Stage
  • 7: Crystalman's Stage
  • 8: Protoman Castle Stage 1
  • 9: Protoman Castle Stage 2
  • A: Protoman Castle Stage 3
  • B: Protoman Castle Stage 4
  • C: Wily Castle Stage 1
  • D: Wily Castle Stage 2
  • E: Wily Castle Stage 3
  • F: Wily Castle Stage 4
