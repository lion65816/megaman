//////////////////////////////////////////////
	ロックマン2用パラメータエディタ
//////////////////////////////////////////////

このエディタは、ロックマン2のステージいじりで
どうもエディタじゃいじりづらいところを寄せ集めたものです。

プログラムスキルは高くないので、致命的なバグがあるかもしれません。
使用する際にはバックアップを忘れずにお願いします。

●できること：
　　　・スクロール設定をドラッグ＆ドロップで変更
　　　・中間地点、着地Y座標の指定
　　　・シャッター関連の変更
　　　・ボスが出現する画面番号の指定
　　　・ステージ固有の地形属性（トゲ、水中判定など）の変更
　　　・ステージパレット、パレットアニメーションの変更
　　　・雑魚敵、ボスのスプライトセットを変更
　　　・雑魚敵のパレットの変更

●使い方など：

左の黒四角の列は画面の配置の仕方を表しています。

白い部分をドラッグ
　　　　図全体をスクロールさせる
黒四角をドラッグ
　　　　スクロール配置を変更する
　　　　前の画面に重ねるようにドロップするとひと続きになる
ホイール
　　　　ズームイン・アウト
右クリック
　　　　図がどこかへ行ってしまったときに配置をリセットさせる


----中間地点の設定
　　　　第一中間地点（一番始め）は変更できません。
　　　　着地Y座標はバイナリ値を設定してるだけなので、分かりにくいかもしれません。

----シャッター設定について
　　　　シャッター演出が始まるようになる部屋の指定です。
　　　　その画面番号からは右スクロールごとにシャッターの演出が入るようになります。
　　　　ボス前辺りにセットするといいでしょう。

　　　　パレットの指定は、背景のパレットセット（4色組）の何番目を使うかの設定です。 
　　　　左側のパレットから順に00, 55, AA, FFを指定します。

----ステージパレットの編集
　　　　パレットはVirtuaNESのパレットを使用しています。　右クリックで色のコピー、
　　　　左クリック時は、選択したパレットにペーストするか、そのパレットの色を変更する
　　　　画面が出ます。

----スプライトバンクセットの設定
　　　　ダイアログ右の画像の使用状況に、#共有あり と出ることがあります。 それは、
　　　　違う敵で同じ画像を共有している印として付けたかったものです。うまく実装できて
　　　　いないので、今のところ深い意味はない表示です。


----バグ、その他
　　　　雑魚敵のパレット設定のリストに同じ雑魚敵が何列も出ますが、どれを選択しても問題ありません。
　　　　シャッターの開始部屋、ボス出現位置の設定がワイリーステージでおかしくなっています。
