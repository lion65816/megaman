ワイリー登場シーンの改造

ワイリーがブルーレットに乗ってやってくるシーンの速度などを設定するアドレスを書こうと
思います。

どうせCXではまるごと書き換えるから使わないんだけどね…



ヘッダの文は自分で+$10してよ？

$EC00からワイリー城のネームテーブル、属性テーブル

$35A23 … 城の額(?)部分のDr.Wマークの縦位置
$35A30 … ↑に+$10した値。　下スクロールが止まったあとの縦位置

$35D6Fから、Dr.Wマークのスプライト情報(5つ分、$14バイト)
4バイトで1スプライト。 例によって
1バイト目 … 縦位置。 ↑の縦位置からの相対座標
2バイト目 … スプライトの種類。 パターンテーブルを見て選んでね、と
3バイト目 … 色と反転。他に書いてある資料と同じのはず。
4バイト目 … 横位置。　こちらは絶対座標


$356DAからイベントが始まります(ちょっと前にBG読み込みなんかがある)

$356DB … 流れる曲。　あのUFOの音は効果音ではなく曲だったのだ！
$356E3 … UFOの初期位置(画面単位)
$356EB … UFOの初期位置(横座標)

$356F3 … UFOの初期縦座標(上半分)
$356F8 … UFOの初期縦座標(下半分)

$3571A … UFOの横速度下位
$35722 … UFOの横速度上位
$3572D … UFOの横速度画面単位　変更すると消えちゃうはず

$3573A … どこまで移動するか
右から左に移動させようとする場合、ちょっと面倒。$3573Bを90にするといいかも。


$35765 … 停止 → 蓋開けまでの時間
$35784 … ワイリー&謎の青いのの横位置
$35789 … ワイリー&謎の青いのの縦位置
$3578E … ワイリー&謎の葵のの画面位置　変更すると消えちゃうはず


$35793 … 蓋を上げる時間
$3579B … 蓋を上げる速度下位
$357A3 … 蓋を上げる速度上位

$357B5 … 眉毛クィッをする時間
$357BF … 眉毛を動かす間隔

$357E4 … 蓋を閉める時間
$357EC … 蓋を閉める速度下位
$357F4 … 蓋を閉める速度上位





後でUFOが振れる速度なんかもあったら追記すると思う。
