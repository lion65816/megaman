簡易説明

ロックマン3のROMイメージを読み込むことで、マップ・オブジェクトの配置等を編集できます。

[パレット]
ここでパレットを作成します。
コンボボックスの定義0〜定義4で編集したいパレットセットを変更します
(ロックマン3ではスクロール毎にどのパレット定義を使用するか決めることが出来ます)。


[16x16チップ]
ここで16x16のチップを作成します。
編集したい16x16チップを選択して、8x8のタイルを組み合わせて下さい。
チップの属性・パレットは、ラジオボタン又はエディットボックスに直接値を入力することで変更できます。
(ラジオボタンでは、チップを属性4bit・パレットを属性2bitとして扱っています。
 本来未使用の残り2bitを使用する場合はエディットボックスに直接値を入力して下さい。)

・16x16チップリスト上の操作
Ctrl + 右クリック … 16x16チップをコピー
Ctrl + 左クリック … 上の操作でコピーした16x16チップを貼り付け
右クリック … マウスカーソルの位置の8x8タイルを選択する

※BGバンクの設定を除き、ワイリーステージ2,3,5でこのデータは共有されています。


[32x32チップ]
ここで32x32のチップを作成します。
編集したい32x32チップを選択して、16x16のチップを組み合わせて下さい。

・32x32チップリスト上での操作
Ctrl + 右クリック … 32x32チップをコピー
Ctrl + 左クリック … 上の操作でコピーした32x32チップを貼り付け
Shift + 右クリック … マウスカーソルの位置の16x16チップを選択する(単に右クリックだけでも可)
Shift + 左クリック … マウスカーソルの位置の16x16チップを選択チップで書き換え

※ワイリーステージ2,3,5でこのデータは共有されています。


[マップセット]
ここで一画面分のマップを作成します(実際に画面をどの順で並べるかは[マップ]タブで編集します)。
32x32チップを組み合わせていきます。
編集対象のチップ上で右クリックすると、そのチップが選択チップになります。
また、[32x32チップ]で編集対象のチップを選択しても、そのチップが選択チップになります。
(選択チップが例のごとく潰れて見にくいので、活用して下さい。)

・マップセット表示画面上での操作
ここで一画面分のマップを作成します(実際に画面をどの順で並べるかは[マップ]タブで編集します)。
32x32チップを組み合わせていきます。
編集対象のチップ上で右クリックすると、そのチップが選択チップになります。
また、[32x32チップ]で編集対象のチップを選択しても、そのチップが選択チップになります。
(選択チップが例のごとく潰れて見にくいので、活用して下さい。)

・マップセット表示画面上での操作
Ctrl + 右クリック … マップセット1画面単位でコピー
Ctrl + 左クリック … マップセット1画面単位で貼り付け
Shift + 左クリック … マップセット編集用グリッドの表示

・マップセット編集用グリッド上での操作
フォーカスのあるセルに対して直接16進数を入力することができます。
セルを選択後、右クリックメニューで、セルの内容をコピー・貼り付けを行うことができます
(貼り付け位置は常にフォーカスのあるセルが左上となります)。

※ワイリーステージ2,3,5でこのデータは共有されています。


[マップ]
マップセットで作った画面を並べていきます。
下の２本のスクロールバーで並べる画面を選択して下さい。


[オブジェクト]
敵やアイテム等のオブジェクトの配置・編集を行います。
・ドラッグでオブジェクトの移動を行います。この際Ctrlキーを押しながらだと8ドット、
　Shiftキーを押しながらだと4ドット単位で移動します。

・Shift+Ctrlキーを押しながらオブジェクトをドラッグしても、オブジェクトの座標は変わりません。
　これにより不意のオブジェクトの移動を防ぐことができます。

・オブジェクトをクリックすると、編集対象オブジェクト欄でオブジェクトの種類を変更できます。

・右クリックでポップアップメニューが表示され、オブジェクトの追加・削除等ができます。

・「オブジェクトの番号を自動変更」をチェックで、X座標の小さいオブジェクト順に番号が振られます。
　チェックしておいた方が無難です。

・Shift + 左クリックでオブジェクト編集用ウィンドウを表示します。


「再読み込み」ボタンについて
各項目について、ステージ毎にROMバッファからデータを読み込み直します。
ファイルの保存処理を行うと、ROMバッファ自体が更新されるので注意して下さい。


「元に戻す/やり直し」について
所謂アンドゥ/リドゥ機能です。現在はオブジェクトの編集を除いた編集に対して機能しています。
それぞれCtrl+Z,Ctrl+Yがショートカットキーとして割り当てられています。

アンドゥ/リドゥのデータは各ステージ毎に独立に管理しています。
また、ROMイメージの保存、再読み込み、重複チップの置換の操作を行うと
そのステージのアンドゥ/リドゥのデータは消えます。
アンドゥ/リドゥ機能を無効にしたい場合は、INIファイルを開き
[Undo]セクションの"Enabled=1"を"Enabled=0"に書き換えて下さい。
